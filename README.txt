Description
===========

A simple program that counts money sent you by spammers.

How to use
==========
1) First, extract the archive to wherenever is convinient. Make sure file named smc is executable.
2) Fill in your credentials in mailbox.config. Example:

[Gmail]
login=john.smith
password=cool.password

3) Run smc from the folder you installed it to:
$ ./smc

4) Observe a new icon appeared in your taskbar. The tooltip will tell you hom much money you received from spammers so far!
